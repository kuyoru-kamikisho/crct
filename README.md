### 介绍

这是一个键盘映射软件，本软件的适合人群可能是数位屏使用者或者类似的人群，
当数位屏较大的时候键盘操作反而不是那么的方便，
本软件可以用来简化甚至替代键盘操作，如果你也面临类似的键盘操作问题，
那么可以从这里开始。

- 支持插入行换行符，用于用户自己去进行排版
- 可调节透明度
- 可设置多套快捷键映射

当前软件发行版本均为Windows版本，其他软件版本可以自行下载项目进行构建。
